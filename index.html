trackerrr [![Build Status](https://travis-ci.org/christophebedard/trackerrr.svg?branch=kinetic-devel)](https://travis-ci.org/christophebedard/trackerrr)

Track tags and other features with an RRR robot manipulator.

Prerequisites

  1. ROS kinetic

Initial setup

  1. Create workspace directory and init the catkin workspace
    1 mkdir -p ~/tracker_ws/src
    2 cd ~/tracker_ws/src
    3 catkin_init_workspace
  2. Clone this repo in the src/ directory
    1 cd ~/tracker_ws/src
    2 git clone https://github.com/christophebedard/trackerrr.git
  3. Get source dependencies with
    1 cd ~/tracker_ws/src
    2 wstool init
    3 wstool merge trackerrr/dependencies.rosinstall
    4 wstool up
    Or clone the following repositories in the src/ directory
    1 cd ~/tracker_ws/src
    2 git clone https://github.com/ROBOTIS-GIT/DynamixelSDK.git
    3 git clone https://github.com/ROBOTIS-GIT/dynamixel-workbench.git
    4 git clone https://github.com/ROBOTIS-GIT/dynamixel-workbench-msgs.git
    5 git clone https://github.com/christophebedard/open_manipulator.git
    6 git clone https://github.com/ROBOTIS-GIT/open_manipulator_msgs.git
  4. Get package dependencies with
    1 cd ~/tracker_ws
    2 rosdep install -y --from-paths src --ignore-src --rosdistro kinetic
    Or install them with
    1 sudo apt-get install libv4l-dev \
    2 ros-kinetic-qt-build \
    3 ros-kinetic-gazebo-ros-control \
    4 ros-kinetic-position-controllers \
    5 ros-kinetic-robotis-math \
    6 ros-kinetic-moveit \
    7 ros-kinetic-camera-umd
    camera_umd (deprecated) is used instead of libuvc_camera because of this

Feature selection

Additionally, clone at least one of these, in order to:

  1. track AprilTags
    1 cd ~/tracker_ws/src
    2 git clone https://github.com/RIVeR-Lab/apriltags_ros.git
  2. track objects with YOLO
    1 cd ~/tracker_ws/src
    2 git clone https://github.com/leggedrobotics/darknet_ros.git
    Follow instructions here.

Build

  1. Compile

    1 cd ~/tracker_ws
    2 catkin build

    To get better performance, compile a release build

    1 cd ~/tracker_ws
    2 catkin build -DCMAKE_BUILD_TYPE=Release
  2. Source
    1 cd ~/tracker_ws
    2 source devel/setup.bash

Launch files

Some of the main launch files:

  • tracker_tag.launch
    • tracks an AprilTag (apriltags_ros needed!)
    • main arguments
      • dof : degrees of freedom of the robot [1]
      • target_tag_id : ID of target tag [27]
  • tracker_yolo.launch
    • tracks an object (darknet_ros needed!)
    • main arguments
      • dof : degrees of freedom of the robot [1]
      • target_object : name/class of target object
  • joy_test.launch
    • drives the robot with a controller (why not)
    • main argument
      • dof : degrees of freedom of the robot [1]
